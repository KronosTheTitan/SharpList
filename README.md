This project was intended as a learning exercise about data structures in C++. While also provding me with a class that I can use in future projects.

functions available on the class are:
 - list()
 - list(const T* items, int n)
 - list(int initial_capacity)
 - ~list()
 - void add(T item)
 - void add_range(T* items, int n)
 - void clear()
 - bool contains(T item)
 - int ensure_capacity(int desired_capacity)
 - int index_of(T item)
 - int index_of(T item, int index)
 - int index_of(T item, int index, int end)
 - void remove(T item)
 - void remove_range(T* items, int n)
 - void remove_at(int index)
 - void reverse()
 - void trim_excess()

The development of this project was a fun learning opportunity about some memory concepts and about templates, a feature of C++ I had not used before.
